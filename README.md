## Install via [npm](https://npmjs.com)

```sh
$ npm install caverphone-phonetics
```

## Usage

```js
var caverphone = require( 'caverphone-phonetics' )
```

```js
caverphone( 'Stevenson' )
// > 'STFNSN1111'
```
